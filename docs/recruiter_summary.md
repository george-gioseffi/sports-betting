# Recruiter Summary

## What This Project Demonstrates
This repo shows end-to-end delivery of an analytics product: ingestion, data quality, KPI modeling, risk governance, simulation, and dashboarding.

## Technical Skills Evidenced
- Python engineering with modular architecture
- SQL modeling and analytical query design
- Layered data architecture (`raw`, `staging`, `marts`)
- KPI design and metric governance (ROI, yield, CLV, drawdown, expectancy)
- Hybrid data design (real match outcomes + simulated pricing/picks)
- Data quality validation and testing discipline
- Scenario simulation and risk scoring logic
- Streamlit dashboard productization
- CI and lint/test automation with GitHub Actions

## Why It Is Relevant for Data Roles
- **Data Analyst / BI Analyst:** business KPI clarity and communication-ready visuals
- **Analytics Engineer:** structured marts, reproducibility, and modeling consistency
- **Junior Data Engineer:** pipeline design, data contracts, and operational discipline
- **Product/Data Analytics:** decision framing with risk-aware interpretation

## Signal to Hiring Teams
The project shows practical ownership, analytical judgment, and product thinking beyond notebook-only analysis.
